# UX Strategy

## Objective

The objective of this phase was to define the structure, navigation, and user journey before moving into wireframing and visual design.

## Deliverables

### User Flow

Designed the complete user journey from onboarding to food ordering and checkout.

### Checkout Flow

Mapped the checkout experience to identify potential friction points and optimize the ordering process.

### Information Architecture

Structured screens, content, and features to improve discoverability and navigation.

### Mood Board

Created a visual direction to establish branding, color palette, typography, and overall design language.
